# EMG-Based Prosthetic Hand

This project implements a prosthetic hand controlled using EMG (muscle) signals in real time.

## Features
- Processes EMG signal data to detect patterns
- Implements control logic for finger movements
- Real-time response based on muscle input

## Tech Stack
- Python
- Arduino
- EMG Sensors

## Description
The system captures muscle signals using EMG sensors and processes the data to control servo motors, enabling real-time prosthetic movement.
